# Changelog

## 1.0.1 - 2026-07-02

### Changed

- Reduced public Markdown clutter by keeping internal notes, generated audit prompts and local workflow documents out of the repository view.
- Added `docs/README.md` as the public documentation entry point.

## 1.0.0 - 2026-07-02

Initial public release of Suno Song Studio.

### Added

- Self-hosted FastAPI backend with React/Vite frontend.
- SunoAPI.org generation, import and follow-up production workflows.
- Local-first audio, cover, lyrics, metadata and task preservation.
- Library views for grouped songs, direct title lists and cover browsing.
- Song details with advanced Suno options, SRT, stems, WAV, cover and export tools.
- Bilingual German/English React UI.
- Local audio analysis and optional AI-assisted production helpers.
- Admin configuration for providers, profiles and optional workflow features.
- Step-by-step installation guide and AGPL-3.0 license.

### Notes

- Real provider keys are intentionally excluded.
- Personal deployment and repository-management scripts are ignored and not part of the public package.
- External provider usage can create costs depending on local configuration.
